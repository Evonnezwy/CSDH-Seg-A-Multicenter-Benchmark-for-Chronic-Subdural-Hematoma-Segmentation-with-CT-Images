# CSDH-Seg-A-Multicenter-Benchmark-for-Chronic-Subdural-Hematoma-Segmentation-with-CT-Images
1. Dataset name: CSDH-Seg
2. Imaging modality: Non-contrast head CT
3. Task: Chronic subdural hematoma segmentation
4. Annotation: voxel-level hematoma mask
5. File format: NRRD
6. Image file: ct.nrrd
7. Mask file: seg_bleeding.nrrd
8. Centers: Center1, Center2, Center3
9. Data split strategy: cases from three Centers were randomly divided into a training set and an internal validation set at a ratio of 7:3.
